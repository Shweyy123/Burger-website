# Burger Website

Welcome to the Burger Website repository! This project is a simple and visually appealing burger-themed website created with HTML, CSS, and JavaScript. It is designed to showcase a burger menu with various sections like Home, Menu, About, Reviews, Contact, and Blogs.

## Features

- **Home Page**: Features a hero section with a delicious burger image as the centerpiece.
- **Menu Section**: Displays different types of burgers with images and descriptions.
- **About Section**: Provides information about the burger restaurant.
- **Reviews Section**: Showcases customer reviews and testimonials.
- **Contact Section**: Contains a form for visitors to reach out to the restaurant.
- **Blogs Section**: Includes burger-related blog posts and articles.

## Technologies Used

- **HTML5**: For structuring the content of the pages.
- **CSS3**: For styling the website and adding visual elements.
- **Bootstrap 5**: For responsive design and layout.
- **JavaScript**: To add interactivity where needed.

Installation :

To run this project locally:

1. Clone the repository using the following command:

       git clone https://github.com/shweyy123/Burger-website.git

2. Navigate to the project directory:

       cd Burger-website


3. Open the index.html file in your web browser to view the website locally.
